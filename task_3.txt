# Docker Image Creation Task

## Task – Create and Run Nginx and Node Docker Applications

```bash
# Nginx Docker App

mkdir nginx-app
cd nginx-app

notepad index.html

notepad default.conf

notepad Dockerfile

docker build -t custom-nginx:v1 .
docker run -d -p 8080:80 --name nginx-container custom-nginx:v1

# Open in browser
http://localhost:8080


# Node Docker App

mkdir node-app
cd node-app

apt update
apt install nano -y

# Create files
nano app.js
nano package.json
nano Dockerfile

# Verify files
dir

# If Dockerfile saved as txt then rename
Rename-Item Dockerfile.txt Dockerfile

# Build Docker image
docker build -t node-demo:v1 .

# Run container
docker run -d -p 3000:3000 --name node-container node-demo:v1

# Check running containers
docker ps

# Open in browser
http://localhost:3000
```
